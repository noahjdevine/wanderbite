# USER STORIES & TECH STACK

## Tech Stack
- **Frontend:** Next.js (App Router), TypeScript, Tailwind.
- **Backend:** Supabase (Auth, DB, Realtime).
- **Maps:** Mapbox or Google Maps API.
- **Payments:** Stripe Subscriptions.

## User Stories
### Epic: Assignment & Guardrails
- As a user, I want to receive 2 assigned challenges each month so I know what to try.
- As a user, I want 1 swap per month so I have a pressure valve if a pick doesn't work.
- As the system, I must enforce cooldown rules (6mo gap, 2x/year cap) to prevent fatigue.

### Epic: Preferences
- As a user, I want to set allergies (Hard Stop) so I am never assigned unsafe food.
- As a user, I want to set a distance band (Close / Worth Trip / Adventure) to fit my lifestyle.

### Epic: Redemption
- As a user, I want to redeem a challenge to unlock the $10 discount.
- As a restaurant, I want to verify a redemption quickly without slowing down service.
